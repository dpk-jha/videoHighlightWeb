# Server
1. install python [any version]
2. install flask
3. cd server
4. flask run

# Client
1. install nodejs
2. cd client
3. npm install

4. Deploy
    1. Development way
        1. npm start
        2. live on 3000 probably

    1. Production way
        1. npm build
        2. cd client/build/
        3. copy all files to template folder form where you can serve them
